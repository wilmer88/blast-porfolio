Taco Stand

My pride and joy. Developed with Stripe checkout payment option, MongoDB Atlas ran on Amazon Web Service. Developed with ES6, Bulma CSS framework, React version 18.0.0, JSX, react-router-dom, JSON, conditional rendering, Axios, NODE, Mongoose, Postman, Express, Bcrypt, jwt, Mocha and Nyc for some testing covarage. process.env, mobile responsiveness, Studio 3T, Google Chrome Dev Tools, React Developer Tools, Heroku, GitHub and VS Code.

Responsive on all modern devicess a MERN Stack, Api driven cloud based (SaaS) for food service trucks or restaurant businessess. The aplication helps customers and buisness owners save time by allowing the customer to place orders at the business location or pre ordering at the customer home or current location thus helping businessess by not slowing down and/or stoping other order preparation processes and by not needing to have anyone there when an order is being placed or paid for.
